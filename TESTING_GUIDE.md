# Руководство по тестированию исправлений совместимости с Ant Design

## Быстрый тест

### 1. Обновите пакет в вашем проекте

```bash
cd /path/to/your/project
npm update @corp/workspace-menu
```

### 2. Проверьте версию

```bash
npm list @corp/workspace-menu
```

Должно быть: `@corp/workspace-menu@1.0.2`

### 3. Запустите проект

```bash
npm run dev
# или
npm start
```

### 4. Тестовые сценарии

#### ✅ Сценарий 1: Dropdown Menu

1. Найдите виджет "IT Сервисы" на странице
2. Кликните на кнопку виджета
3. **Ожидаемый результат**: Выпадающее меню должно открыться
4. Проверьте, что меню отображается поверх всех элементов Ant Design
5. Кликните на любой пункт меню
6. **Ожидаемый результат**: Переход должен работать, меню должно закрыться

#### ✅ Сценарий 2: Modal "Все сервисы"

1. Откройте dropdown меню виджета
2. Кликните на "Все сервисы"
3. **Ожидаемый результат**: Модальное окно должно открыться
4. Проверьте, что модальное окно отображается поверх всех элементов Ant Design
5. Проверьте, что можно прокручивать содержимое
6. Кликните на любую карточку приложения
7. **Ожидаемый результат**: Переход должен работать

#### ✅ Сценарий 3: Одновременная работа с Ant Design

1. Откройте любое модальное окно Ant Design в вашем приложении
2. Не закрывая его, попробуйте открыть dropdown виджета
3. **Ожидаемый результат**: Dropdown должен открыться поверх модального окна Ant Design
4. Попробуйте открыть "Все сервисы"
5. **Ожидаемый результат**: Модальное окно виджета должно отобразиться поверх всего

#### ✅ Сценарий 4: Overlay и закрытие

1. Откройте модальное окно "Все сервисы"
2. Кликните на затемненную область (overlay)
3. **Ожидаемый результат**: Модальное окно должно закрыться
4. Откройте снова и нажмите кнопку закрытия (X)
5. **Ожидаемый результат**: Модальное окно должно закрыться

## Проверка в DevTools

### 1. Проверьте DOM структуру

Откройте DevTools (F12) → Elements и найдите:

```html
<body>
  <!-- Ваше приложение -->
  <div id="root">...</div>

  <!-- Должен быть изолированный контейнер виджета -->
  <div
    id="workspace-menu-portal-root"
    data-workspace-menu-portal-root
    style="...z-index: 2000..."
  >
    <!-- Portal контент -->
  </div>
</body>
```

### 2. Проверьте z-index в инспекторе

1. Откройте dropdown или modal виджета
2. В DevTools выберите элемент
3. Проверьте Computed стили:
   - Portal root: `z-index: 2000`
   - Dialog content: `z-index: 2001`
   - Dropdown: `z-index: 2000`

### 3. Проверьте изоляцию стилей

1. Откройте modal виджета
2. В DevTools найдите элементы с `[data-workspace-menu-portal]`
3. Проверьте, что не применяются глобальные стили Ant Design
4. Проверьте свойство `isolation: isolate` на `.workspace-menu-root`

## Частые проблемы и решения

### Проблема: Dropdown не открывается

**Решение:**

1. Проверьте версию пакета: должна быть 1.0.2+
2. Очистите кеш: `rm -rf node_modules/.cache`
3. Пересоберите: `npm run build`
4. Перезапустите dev-сервер

### Проблема: Modal скрыт под Ant Design компонентами

**Решение:**

1. Проверьте в DevTools, что контейнер `#workspace-menu-portal-root` существует
2. Проверьте z-index контейнера (должен быть 2000)
3. Убедитесь, что в вашем проекте нет кастомных z-index > 2000

### Проблема: Клики не работают

**Решение:**

1. Проверьте в консоли браузера на наличие ошибок
2. Убедитесь, что pointer-events не отключены
3. Проверьте, что на элементы не наложены другие overlay

### Проблема: Стили виджета сломались

**Решение:**

1. Убедитесь, что импортированы стили виджета:
   ```tsx
   import "@corp/workspace-menu/dist/index.css";
   ```
2. Проверьте, что Tailwind CSS настроен корректно
3. Проверьте, нет ли конфликтов с вашими глобальными стилями

## Автоматизированное тестирование

Если у вас есть E2E тесты (Playwright, Cypress):

```javascript
// Пример теста
test("widget dropdown opens over Ant Design modal", async ({ page }) => {
  // Открыть Ant Design modal
  await page.click('[data-testid="open-ant-modal"]');

  // Открыть widget dropdown
  await page.click('[data-testid="workspace-menu-trigger"]');

  // Проверить, что dropdown виден
  const dropdown = await page.locator('[data-slot="dropdown-menu-content"]');
  await expect(dropdown).toBeVisible();

  // Проверить z-index
  const zIndex = await dropdown.evaluate(
    (el) => window.getComputedStyle(el).zIndex
  );
  expect(parseInt(zIndex)).toBeGreaterThan(1000);
});
```
