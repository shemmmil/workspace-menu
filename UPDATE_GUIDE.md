# Инструкция по обновлению виджета для совместимости с Ant Design

## Проблема решена! ✅

Виджет теперь полностью совместим с Ant Design и корректно отображается поверх всех компонентов.

## Как обновить в вашем проекте

### Шаг 1: Обновите пакет

В директории вашего проекта, где используется виджет, выполните:

```bash
npm update @corp/workspace-menu
```

или

```bash
npm install @corp/workspace-menu@latest
```

### Шаг 2: Пересоберите проект

```bash
npm run build
```

или для dev-режима:

```bash
npm run dev
```

### Шаг 3: Проверьте работу

1. Откройте ваше приложение
2. Кликните на виджет меню
3. Убедитесь, что выпадающее меню отображается корректно
4. Откройте модальное окно "Все сервисы"
5. Проверьте, что модальное окно отображается поверх всех элементов Ant Design

## Что было исправлено

### Изменения z-index:

- **Dialog overlay**: теперь `2000` (было `50`)
- **Dialog content**: теперь `2001` (было `50`)
- **DropdownMenu**: теперь `2000` (было `50`)

### Добавлена изоляция CSS:

- Специальный атрибут `data-workspace-menu-portal` для изоляции порталов
- CSS переменная `--workspace-menu-z-index` для гибкой настройки
- Сброс box-sizing для предотвращения конфликтов стилей

## Сравнение z-index

| Компонент            | Старое значение | Новое значение |
| -------------------- | --------------- | -------------- |
| Dialog Overlay       | 50              | **2000**       |
| Dialog Content       | 50              | **2001**       |
| DropdownMenu Content | 50              | **2000**       |

vs Ant Design:

| Компонент Ant Design | z-index |
| -------------------- | ------- |
| Modal                | 1000    |
| Dropdown             | 1050    |
| Tooltip              | 1060    |

## Обратная совместимость

✅ Эти изменения **полностью обратно совместимы**  
✅ Не требуется изменений в коде вашего приложения  
✅ Виджет будет работать как в проектах с Ant Design, так и без него

## Если у вас кастомные z-index > 2000

Если в вашем проекте есть компоненты с z-index выше 2000, вы можете переопределить значение для виджета:

```css
/* Добавьте в ваш глобальный CSS файл */
[data-workspace-menu-portal] {
  --workspace-menu-z-index: 3000; /* или другое нужное значение */
}
```

## Поддержка

Если после обновления возникли проблемы:

1. Убедитесь, что версия пакета 1.0.1 или выше
2. Очистите кеш сборки: `rm -rf node_modules/.cache`
3. Пересоберите проект: `npm run build`
4. Проверьте консоль браузера на наличие ошибок

Если проблема сохраняется, создайте issue с описанием проблемы и скриншотом.
